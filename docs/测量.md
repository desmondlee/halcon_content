# 测量

工业测量的基础——卡尺工具。

## 卡尺工具过程

1. 确定边缘检测方向；
2. 高斯滤波；
3. 执行投影处理；
4. 执行微分（对比度）处理；
5. 设置边缘敏感度（对比度阈值）、边缘极性、边缘的位置；
6. 边缘执行亚像素处理。
