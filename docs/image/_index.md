### 通用流程
{{< mermaid>}}

B --> |有| C[获得位置,角度信息]
B --> |无| A

### 常用标定

1. 像素变换(原型、矩形、像素单量)；
2. 透视投影及光学畸变校正；
3. 不同坐标系之间的标定(三点确定一个坐标系)，如九点定标(机械手)；
4. 多个相机通过标定统一到同一个坐标系。
